**FPSLite** - Spoof your device as different model for specific games to unlock higher FPS.

---

## Features:

- **Device model spoofing** for better game compatibility.  
- Unlocks **60/90/120 FPS** in supported games.
---

## Installation:

1. Download the latest release from the [Releases](../../releases) page  
2. Open **Magisk/KernelSU Manager**  
3. Tap on **Modules** > **Install from storage**  
4. Select the downloaded `.zip` file  
5. **Reboot Device** your device
